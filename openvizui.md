请帮我开发一个名为 OpenVizUI 的桌面应用程序，它是一个可视化 CLI 工具管理平台。

项目目标：为 Claude Code、Gemini、OpenCode 和 OpenClaw 提供现代化图形界面，让用户无需记忆复杂的命令行操作。

核心需求：
1. 多工具集成 - 统一管理四个AI开发工具
2. 环境检测 - 自动检测系统环境并引导配置
3. 一键安装卸载 - 提供自动化安装和清理功能
4. 可视化配置 - API Key、代理、模型等图形化配置
5. 现代化界面 - 深色/浅色主题，10语言国际化
6. 聊天界面美化 - 可自定义颜色主题的聊天界面
7. 安全存储 - API Key等敏感信息加密存储

技术栈建议：React + TypeScript + Tauri，支持 Windows/macOS/Linux。

🎯 核心功能需求
1. 多工具集成管理
    支持四大 CLI 工具：
        Claude Code（Claude CLI）
        Google CLI（Gemini API）
        OpenCode
        OpenClaw
    统一仪表盘展示所有工具状态
    一键切换不同工具界面

2. 智能环境检测与自动化
    前置依赖检测：
        Node.js & npm 版本检查
        Git 安装状态
        Python 环境（如有需要）
    工具目录检测：
        ~/.claude/
        ~/.gemini/ 或 ~/.google-cli/
        ~/.opencode/
        ~/.openclaw/
    自动化修复：提供一键安装缺失依赖

3. 一键安装/卸载管理
    每个工具提供：
        安装按钮（自动执行正确命令）
        卸载按钮（彻底清理）
        更新检查与一键更新
        版本管理（支持多个版本切换）

4. 配置可视化编辑器
    统一配置界面：
        API Key 管理（加密存储）
        代理设置（HTTP/HTTPS/SOCKS5）
        模型选择（根据工具支持）
        超时设置、重试策略
    工具特定配置：
        MCP（Model Context Protocol）服务器管理
        Skills/插件管理
        上下文长度、温度等参数

5. 现代化用户界面
    设计系统：
        浅色/深色主题切换
        现代化组件库（建议使用：Ant Design、MUI、Shadcn/ui）
        交互动画与微交互
    布局：
        左侧导航栏（工具切换）
        主内容区（当前工具操作）
        右侧面板（配置/状态）
        底部状态栏（系统信息）

6. 国际化支持
    支持10种语言：
        英语、简体中文
        日语、韩语、意大利语
        法语、德语、西班牙语
        俄语、阿拉伯语
    本地化存储：用户语言偏好保存

7. 聊天界面增强
    个性化聊天：
        消息气泡颜色自定义
        预置主题包（Material、Nord、Dracula、Solarized等）
        自定义 CSS 支持
    功能：
        聊天历史记录
        会话管理
        导出/导入聊天记录
        代码语法高亮

8. 安全特性
    API Key 加密存储（系统密钥链）
    配置备份与恢复
    操作审计日志
    敏感信息隐藏选项

9. 高级功能
    工作区管理：
        项目特定的配置预设
        快速切换不同项目环境
    自动化脚本：
        录制操作生成脚本
        定时任务设置
    监控与诊断：
        实时查看 CLI 输出
        网络连接测试
        性能监控
